# To set up Google OAuth:
# 1. Go to Google Cloud Console > APIs & Services > Credentials
# 2. Create OAuth 2.0 Client ID (Web application)
# 3. Add authorized JavaScript origins:
#    - http://localhost:5173 (for development)
#    - https://your-app.vercel.app (for production)
# 4. Copy the Client ID to VITE_GOOGLE_CLIENT_ID

# To set up Vercel Postgres:
# 1. Go to your Vercel project dashboard
# 2. Go to Storage tab
# 3. Create a new Postgres database
# 4. The environment variables will be automatically added

# To set up Stripe:
# 1. Go to https://dashboard.stripe.com/register (create account or login)
# 2. Make sure "Test mode" is enabled (toggle in top right)
# 3. Go to Developers > API keys
# 4. Copy "Publishable key" to VITE_STRIPE_PUBLISHABLE_KEY
# 5. Copy "Secret key" to STRIPE_SECRET_KEY
# 6. For webhooks (optional, for production):
#    - Go to Developers > Webhooks
#    - Add endpoint: https://your-app.vercel.app/api/stripe/webhook
#    - Select events: checkout.session.completed, payment_intent.payment_failed
#    - Copy webhook signing secret to STRIPE_WEBHOOK_SECRET
#
